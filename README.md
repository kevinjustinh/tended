# Tended 🐾

> A warm, Notion-inspired iOS app for pet owners to track daily care tasks, health records, and routines for their dogs and cats.

🚧 **Status:** In Development

---

## About

Built as a personal project to solve a real problem — keeping track of multiple pets' feeding schedules, medications, and vet appointments in one place. Tended gives pet owners a single, calm, and beautiful home for everything their pets need, every day.

## Features

- **Today tab** — Daily task list grouped by category, with a completion ring, date navigation, and filters (All, Upcoming, Overdue, Done)
- **Pets tab** — Pet profiles with photos, daily routine, health stats, and notes
- **Tasks tab** — All tasks across all pets, grouped by category and sorted chronologically
- **Calendar tab** — Appointment view for vet visits and grooming
- **Settings** — Notification and display preferences

---

## Tech Stack

- Swift + SwiftUI (iOS 17+)
- SwiftData for local persistence
- MVVM + Repository architecture
- Local notifications via `UserNotifications`
- SF Symbols 5 + Lottie for animations

---

## Requirements

- Xcode 15+
- iOS 17.0 deployment target

---

## Getting Started

1. Clone the repo
```bash
git clone https://github.com/kevinjustinh/tended.git
```
2. Open `Tended/Tended.xcodeproj` in Xcode
3. Select a simulator or device and run

No external dependencies or API keys required.
