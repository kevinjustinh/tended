# Tended

A warm, Notion-inspired iOS app for pet owners to track daily care tasks, health records, and routines for dogs, cats, and other pets.

## Features

- **Today tab** — Daily task list grouped by category, with a completion ring, date navigation, and filters (All, Upcoming, Overdue, Done)
- **Pets tab** — Pet profiles with photos, daily routine, health stats, and notes
- **Tasks tab** — All tasks across all pets, grouped by category and sorted chronologically
- **Calendar tab** — Appointment view for vet visits and grooming
- **Settings** — Notification and display preferences

## Tech Stack

- Swift + SwiftUI (iOS 17+)
- SwiftData for persistence
- Local notifications via `UserNotifications`

## Requirements

- Xcode 15+
- iOS 17.0 deployment target

## Getting Started

1. Clone the repo
2. Open `Tended/Tended.xcodeproj` in Xcode
3. Select a simulator or device and run

No external dependencies or API keys required.
